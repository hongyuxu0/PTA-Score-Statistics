# PTA成绩统计工具
通用PTA成绩单统计工具，基于tkinter+Pandas实现，支持批量处理PTA成绩单、自定义得分率-分数段规则、汇总统计分总分。

## 功能介绍
- 支持批量导入CSV/Excel格式的PTA成绩单
- 自动识别文件编码，解决中文乱码问题
- 按得分率适配不同满分的成绩单统计
- 批量生成带统计分的文件，并汇总所有学生的统计分总分

## 使用方法
1. 运行工具：双击打包后的`PTA成绩统计工具.exe`（或本地运行`通用成绩统计工具.py`）
2. 选择文件：上传PTA导出的成绩单文件
3. 配置规则：设置得分率-分数段对应规则
4. 开始统计：生成带统计分的文件，最后汇总总分
